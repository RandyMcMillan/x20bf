## Getting Started

##### [git](https://git-scm.com/downloads)

```
git clone https://github.com/0x20bf-org/0x20bf.git
cd 0x20bf
```

##### [python@3.8+](https://www.python.org/downloads/)

```
git clone https://github.com/0x20bf-org/0x20bf.git
cd 0x20bf
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install -e .
```
